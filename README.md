# Tableau Employment Dashboard Project

This project presents an interactive Tableau dashboard created during a **Data Technician Bootcamp**, focusing on **employment data analysis**. The dashboard highlights employment trends in the UK by industry type, location, and job availability through clear, interactive visualizations.

## Key Features

- **Interactive Dashboards:** Explore employment statistics across industries and observe changes in employment across the UK.  
- **Filters & Slicers:** Dynamically select cities on an interactive map to view job availability and industry-specific data.  
- **Variety of Charts:**
  - **Bar Charts:** Compare employment figures across sectors.  
  - **Maps:** Geographically represent average changes in employment by city.  
  - **Tree Maps:** Display the top 10 sub-industries by employment.  
  - **Interactive Tables:** Highlight the number of available jobs per city over selected time periods.  
- **Data Storytelling:** Visualizations are organized to guide the audience through key labor insights and workforce trends.
